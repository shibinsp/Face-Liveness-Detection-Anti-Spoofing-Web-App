## üéØ Quick Reference

### Installation Commands

```bash
# Quick install (2 minutes)
pip install streamlit opencv-python numpy pandas
streamlit run app_antispoofing.py

# Full install (30 minutes)
pip install -r requirements.txt
streamlit run app.py
```

### Key Concepts

- **Anti-Spoofing:** Detecting fake faces (photos, videos, masks)
- **Liveness Detection:** Verifying the face is from a live person
- **Texture Analysis:** Computer vision-based detection (fast)
- **ONNX Models:** Deep learning-based detection (accurate)
- **Multi-Face:** Detecting multiple faces simultaneously
- **Streaming:** Real-time continuous detection

### Performance Metrics

- **Texture Analysis:** 80-85% accuracy, 15-20 FPS
- **ONNX Models:** 95%+ accuracy, 10-15 FPS CPU, 30+ FPS GPU
- **InsightFace:** 95%+ accuracy, 10-15 FPS CPU, 40-60 FPS GPU
